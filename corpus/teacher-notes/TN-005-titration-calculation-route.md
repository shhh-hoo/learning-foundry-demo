# Titration calculation route

A titration problem combines experimental-data selection with stoichiometry.

## Stage 1: select the titre

- exclude the rough titre;
- identify concordant titres using the stated school/exam rule;
- calculate the mean from the accepted set;
- preserve the selected readings as evidence.

## Stage 2: calculate

```text
known concentration × mean volume
→ known amount
→ equation ratio
→ unknown amount
→ requested concentration, mass, purity or formula
```

Convert `cm³` to `dm³` before using `n = cV`.

## Common errors

- averaging every reading;
- using the rough titre;
- using `cm³` directly in `n = cV`;
- reversing the equation ratio;
- stopping at the concentration in the flask when the question asks for the original solution;
- missing a dilution or aliquot factor.
