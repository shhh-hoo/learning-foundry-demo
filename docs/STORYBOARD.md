# Storyboard

The canonical product story is [`demo-storyboard/storyboard.md`](../demo-storyboard/storyboard.md). Each 1920×1080 image captures the complete Demo Shell around a real interactive product frame; no product text is composited afterward.

Run `npm run demo:local` and then `npm run storyboard` to reproduce all seven frames. The Playwright journey resets the local session, completes all six scenes, verifies the strengthened runtime hint and finally opens Engineering Inspector.
