# Case study

The structural repair replaces a prewritten route with a real DeepSeek Agent and explicit tool loop. Retrieval is source-grounded, MASS diagnosis crosses the Trainer Diagnosis API, and unsupported multi-stage work produces a bounded capability gap or a request for missing evidence.

Product state starts empty. A configurable `componentId + failureCode` repeated-diagnosis signal can invite teacher review after the configured PRODUCT-run threshold; it is not mature automated Learning Pattern Analysis. A teacher must judge impact, severity, support gaps and instructional value before creating a candidate. Component Contract Checks, named Expert Review and Registry publication remain separate gates.

Kp is retained only for legacy regression and engineering inspection. The archived MP4 and storyboard predate the real Agent and are not canonical release evidence.
