# Demo Shell

The Demo Shell hosts a real product iframe beside an external explanation panel. Guided Story contains six scenes; Free Explore can open any product surface or Standard Trainer without step gates.

Guided navigation is event-gated. Next remains disabled until the current scene's required typed events have been received from the expected iframe window and origin. The Shell runtime-validates each event and ignores malformed or unknown messages. It never calls diagnosis, aggregation, governance or publication functions and never writes product state on behalf of a user action.

The six scenes are learner diagnosis, preserved learning, pattern threshold, candidate governance, registry publication and improved downstream runtime behavior. Engineering Inspector is captured as a seventh supporting frame.
